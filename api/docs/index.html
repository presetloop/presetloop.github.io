API Documentation

Base URL:
  • Tracks: https://api-music.presetloops.com/tracks
    • Returns all track information fields/the entire data set.

How to use

Response Fields:
  • id (Integer): Unique identifier for the track.
  • trackName (String): The name of the track.
  • albumName (String): The name of the album.
  • artistName (String): The artist's name.
  • trackNumber (Integer): Track number on the album.
  • releaseYear (Integer): The release year.
  • trackDuration (String): Duration of the track (e.g. mm:ss).

Query Parameters (where n = a numerical value)

Pagination:

  • /tracks/n Get a track by id.
  • /tracks?_limit=n Limits the number of results returned.
  • /tracks?_start=n Defines the starting index for fetching data.
  • /tracks?_start=n&_limit=n Defines the pagination range.

Sorting:

  • /tracks?_sort=releaseYear Specifies the field to sort by (e.g. trackDuration, releaseYear).
  • /tracks?_order=desc Specifies the sorting order (asc or desc/reverse).

Filtering:

  • /tracks?minDuration=120&maxDuration=300 Filter by track duration.
  • /tracks?genre=New Wave Filter match exact values (e.g. genre=New Wave).

Search:

  • /tracks?_search=vanilla Full-text search on albumName and trackName.

Random:

  • /tracks?_random=5 Fetches a random subset of the dataset.

Combine queries (use &):

  • /tracks?_sort=releaseYear&_order=desc Fetch tracks by release year in descending order.

    • Example Requests:

    • GET https://api-music.presetloops.com/tracks/42
    • GET https://api-music.presetloops.com/tracks?_limit=50
    • GET https://api-music.presetloops.com/tracks?_start=200&_limit=50
    • GET https://api-music.presetloops.com/tracks?minDuration=120&maxDuration=300&_order=desc&genre=Pop
